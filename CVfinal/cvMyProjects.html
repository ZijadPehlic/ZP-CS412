Balls and bins
Balls and bins project is project that was made for Discrete Mathematics course. It is program that tackles probabbilty and statistics. Idea of this project is that user inputs number of balls that will be thrown, and number of bins that balls will be thrown in. Balls are thrown randomly and program outputs following information:
  • Number of empty bins
  • More than n/m bins
  • More than 2n/m bins
  • average empty bins
  • max empty bins
  • Min empty bins
